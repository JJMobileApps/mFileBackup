# 📱 MyPhotoCloud - 사업계획서

## 1. 사업 개요

### 1.1 프로젝트 명
**MyPhotoCloud** - 스마트폰 미디어 자동 백업 & 웹 갤러리

### 1.2 사업 목적
- 스마트폰의 사진/동영상을 자동으로 개인 서버에 백업
- PhotoPrism과 유사한 웹 기반 갤러리 제공
- 개인정보 보호 중시 (클라우드 의존성 제거)
- 가족/팀 내 미디어 공유 플랫폼

### 1.3 타겟 시장
- **주 타겟**: 개인정보 보호를 중시하는 기술 친화적 사용자
- **부 타겟**: 
  - 가족 사진/동영상 관리를 원하는 가정
  - 소규모 팀/동호회
  - NAS 사용자
  - 사진작가/크리에이터

### 1.4 시장 규모 및 기회
- 글로벌 클라우드 스토리지 시장: 연평균 22.3% 성장 (2023-2030)
- 개인정보 보호 우려로 인한 자가 호스팅 솔루션 수요 증가
- PhotoPrism (50만+ 다운로드), Immich 등 오픈소스 대안 인기 급상승

## 2. 제품 및 서비스

### 2.1 핵심 기능

#### 2.1.1 Android 앱 (클라이언트)
1. **자동 백업**
   - 사진/동영상 자동 감지 및 백업
   - Wi-Fi 전용, 충전 중 백업 옵션
   - 중복 파일 자동 건너뛰기
   - 백업 진행률 실시간 표시

2. **스마트 관리**
   - 백업 완료된 파일 자동 삭제 옵션
   - 백업 히스토리 및 로그
   - 프로필별 백업 설정

3. **빠른 접근**
   - 앱 내 미디어 미리보기
   - 웹 갤러리 바로가기

#### 2.1.2 서버 (백엔드)
1. **백업 서버**
   - Kotlin 기반 경량 HTTP 서버
   - REST API (백업, 조회, 삭제)
   - 청크 업로드 지원 (대용량 파일)
   - 중복 제거 (해시 기반)

2. **웹 갤러리**
   - 반응형 웹 인터페이스
   - 사진/동영상 타임라인 뷰
   - 앨범/태그 관리
   - 슬라이드쇼
   - 다운로드/공유 기능

3. **메타데이터 관리**
   - EXIF 데이터 자동 추출
   - 위치 정보 기반 지도 뷰
   - 얼굴 인식 (선택)
   - AI 기반 자동 태깅 (선택)

## 3. 시장 분석

### 3.1 경쟁사 분석

#### Google Photos
- **강점**: 무료 (압축), AI 기능, 편의성
- **약점**: 개인정보 우려, 용량 제한, 종속성
- **대응**: 완전한 프라이버시, 무제한 용량 (자가 서버)

#### PhotoPrism
- **강점**: 강력한 기능, 오픈소스
- **약점**: 복잡한 설치, 모바일 앱 없음
- **대응**: 간편한 설치, 네이티브 Android 앱

- **강점**: 모바일 앱, 직관적 UI
- **약점**: Docker 필수, 초기 설정 복잡
- **대응**: 단일 APK 서버, 간단한 설정
