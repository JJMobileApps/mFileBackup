Account Deletion / 계정 삭제 안내

mFileBackup Account Deletion

This page explains how users can request deletion of their mFileBackup account and related data. Korean and English guidance is provided together for app store review and user support.

Effective date: June 9, 2026
App name: mFileBackup
Developer: JJMobileApps

Deleting an account may also remove access to backed-up data, user-specific folders, and related server records, depending on how the server administrator manages the server.

한국어

1. 계정 삭제 요청 방법

mFileBackup의 사용자 계정은 앱 개발자가 아니라 사용자가 운영하는 서버 관리자에 의해 생성되고 관리됩니다. 계정 삭제가 필요한 경우 해당 서버 관리자에게 직접 요청해야 합니다.

2. 삭제 대상 정보

계정 삭제 요청 시 서버 관리자는 다음 항목을 삭제하거나 접근 불가 상태로 변경할 수 있습니다.

  • 사용자 로그인 계정 정보
  • 사용자별 접근 권한 및 인증 정보
  • 사용자별 백업 폴더 연결 정보
  • 사용자와 연관된 일부 서버 기록 또는 메타데이터

3. 삭제되지 않을 수 있는 정보

서버 운영 방식에 따라 공용 백업 데이터, 시스템 로그, 법적 또는 보안상 보관이 필요한 일부 기록은 즉시 삭제되지 않을 수 있습니다. 이러한 경우에도 해당 계정으로의 추가 접근은 차단될 수 있습니다.

4. 처리 기간

계정 삭제 요청은 서버 관리자 또는 운영 환경에 따라 처리 시간이 달라질 수 있습니다. 일반적으로 합리적인 기간 내에 검토 및 처리됩니다.

5. 문의

계정 삭제와 관련된 문의는 해당 서버 관리자에게 직접 요청해 주세요.

English

1. How to request account deletion

mFileBackup accounts are created and managed by the server administrator, not by the app developer. If you want your account deleted, you should request deletion directly from your server administrator.

2. Data that may be deleted

When an account deletion request is processed, the server administrator may delete or disable access to:

  • User login account information
  • User access permissions and authentication data
  • User-specific backup folder mappings
  • Some server records or metadata associated with the user

3. Data that may not be deleted immediately

Depending on how the server is managed, shared backup data, system logs, or records required for legal or security reasons may not be removed immediately. Even in those cases, further access by the deleted account may be disabled.

4. Processing time

Account deletion requests may take time depending on the server administrator and operating environment. Requests are generally reviewed and handled within a reasonable period.

5. Contact

For account deletion requests or related questions, contact your server administrator directly.
